# iCoder - Heaven for Programmers 🚀

iCoder is a responsive blog website built using **Bootstrap 4**.  
It is designed for coding enthusiasts and developers to read articles about technology, web development, and programming trends.

## 📌 Features

- Responsive Navbar with Dropdown
- Login & Signup Modal Forms
- Bootstrap Carousel with Caption
- Blog Post Cards Layout
- Responsive Grid System
- Footer Section
- Mobile-Friendly Design

## 🛠 Technologies Used

- HTML5
- Bootstrap 4
- CSS (Bootstrap Utility Classes)
- JavaScript (Bootstrap JS)

- Popper.js

3. Open `index.html` in your browser
